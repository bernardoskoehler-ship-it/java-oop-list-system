# 📝 v1 - To-Do List

A simple task list system developed in Java as part of my Object-Oriented Programming (OOP) studies.

---

## 📌 Overview

This project is a basic to-do list application built with Java using `ArrayList`.

It allows the user to:

- add tasks
- remove tasks by position
- list all registered tasks

The main purpose of this project is to practice programming logic, list manipulation, and the first concepts of code organization with Java.

---

## ⚙️ Features

- Add new tasks
- Remove tasks by index
- List all tasks with their positions
- Basic task structure using a dedicated class

---

## 🧠 Concepts Used

- Java
- Object-Oriented Programming (OOP)
- Classes and objects
- `ArrayList`
- Methods
- Loop with `for-each`
- Basic validation logic
