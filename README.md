# 💻 Java OOP List System

This repository contains small projects developed to practice **Object-Oriented Programming (OOP)** concepts in Java.

The focus is on building simple systems using lists, while applying good practices such as:

- Encapsulation
- Validation
- Code organization
- Responsibility separation

---

## 📂 Projects

### 📦 v1-inventory-system

A basic inventory management system.

**Features:**
- Add products
- Remove products
- Update quantities
- Input validation

**Concepts used:**
- Classes and objects
- ArrayList
- Encapsulation
- Basic business rules

---

### 📝 v1-to-do-list

A simple to-do list system.

**Features:**
- Add tasks
- Remove tasks
- List tasks
- Basic validation

**Concepts used:**
- ArrayList
- Control flow
- Method organization
